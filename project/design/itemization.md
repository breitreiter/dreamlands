# Itemization Reference

Goal: the player can reach **+6 from gear** on every check type.

## Gear Bonus Framework

From `dice_mechanics.md` — each check type has a fixed gear formula:

- **Combat** — big: Weapon, pack: —, token: yes
- **Cunning** — big: Armor, pack: —, token: yes
- **Negotiation** — big: —, pack: two pack items, token: yes
- **Bushcraft** — big: —, pack: two pack items, token: yes
- **Mercantile** — big: —, pack: two pack items, token: yes
- **Foraging** — big: Weapon, pack: —, token: yes
- **Injury resist** — big: Armor, pack: —, token: yes
- **Exhausted resist** — big: Boots, pack: one pack item, token: yes
- **Poison resist** — big: Armor, pack: —, token: yes
- **Freezing** — big: —, pack: two pack items, token: yes
- **Thirsty** — big: —, pack: two pack items, token: yes

**Big gear** (Weapon, Armor, Boots) scales +1 to +5.
**Pack gear** comes in exactly two sizes: one +3 item and one +2 item per check type.
**Tokens** are always +1. One per check type. Earned from dungeons.

## Coverage Summary

- **Combat** — big: +4 (need 5), pack: —, token: none → max +4. Missing +5 weapon, token.
- **Cunning** — big: none, pack: —, token: none → max 0. Empty.
- **Negotiation** — big: —, pack: +3/+1*, token: +1 → max +5. Need +2 pack item.
- **Bushcraft** — big: —, pack: none/+2, token: none → max +2. Need +3 pack item, token.
- **Mercantile** — big: —, pack: none/+2, token: none → max +2. Need +3 pack item, token.
- **Foraging** — big: +4 (need 5), pack: —, token: none → max +4. Missing +5 weapon, token.
- **Exhausted resist** — big: none, pack: need +3, token: none → max 0. Empty.
- **Injury resist** — big: none, pack: —, token: none → max 0. Empty (armor has no bonus).
- **Poison resist** — big: none, pack: —, token: none → max 0. Empty (armor has no bonus).

*Writing Kit gives Negotiation +1, not +2.
