# Blackivy Manse

**Biome:** Forest  
**Tier:** 1 (Safe)

## Description

A nobleman's estate from a kingdom no one remembers, its walls intact but entirely sheathed in dark ivy. The ivy does not like fire. The interior is pristine, meals laid out on the table, still warm.
