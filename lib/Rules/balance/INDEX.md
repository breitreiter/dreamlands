# Balance Overview - Quick Reference

## Core Numbers at a Glance

### Character
- Starting Health: **20**
- Starting Spirits: **20**
- Starting Gold: **50**
- Starting Inventory: **10 slots**
- Total Starting Skill Points: **10** (distributed across 10 skills)

### Daily Consumption
- Food: **1/day** (+1 in mountains/swamps)
- Water: **1/day** (+1 in mountains/swamps)

### Travel
- Base Speed: **3 hexes/day** (fixed, not terrain-variable)
- Speed Penalty: **-1** per condition (max -2)
